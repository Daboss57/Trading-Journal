# Trading Journal Wireframes

**Date:** 2025-11-18  
**Purpose:** Low-fidelity wireframes for Day Trading Paper Trading & Backtesting Platform

## Contents

1. **Desktop Views**
   - `01_dashboard_desktop.md` - Main trading dashboard with chart, order entry, positions
   - `02_watchlist_desktop.md` - Watchlist panel with quick actions
   - `03_backtest_builder_desktop.md` - Strategy configuration and backtest setup
   - `04_backtest_results_desktop.md` - Performance metrics and equity curve
   - `05_performance_analytics_desktop.md` - Trade history and analytics dashboard

2. **Mobile Views**
   - `06_dashboard_mobile.md` - Mobile-responsive trading view
   - `07_watchlist_mobile.md` - Mobile watchlist with swipe actions
   - `08_backtest_mobile.md` - Mobile backtest interface

3. **Component Details**
   - `09_order_entry_component.md` - Order ticket detailed specs
   - `10_market_status_header.md` - Market hours indicator component

## Design Principles

- **Speed First:** Minimize clicks for common actions (buy/sell/short)
- **Information Density:** Balance between data-rich and clean
- **Mobile-First Responsive:** Key features accessible on all devices
- **Real-Time Updates:** Live prices, P&L, and positions
- **Visual Hierarchy:** Critical info (account balance, market status) prominent
