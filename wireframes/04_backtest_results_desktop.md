## Component Details

### Header Actions
- **Back to Builder:** Return to backtest configuration (preserve settings)
- **Re-run:** Execute backtest again with same params
- **Save:** Store results to user's backtest history
- **Export CSV:** Download trade log with all details
- **Share:** Generate public link (Phase 2)

### Key Metrics Cards
- **Total Return:** $ profit and % gain from initial capital
- **Total Trades:** Count of executed trades
- **Win Rate:** % winners, count breakdown
- **Profit Factor:** Gross profit / gross loss (>1 = profitable)
- **Sharpe Ratio:** Risk-adjusted return (>1 = good, >2 = excellent)
- **Max Drawdown:** Largest peak-to-trough decline ($ and %)
- **Avg Win/Loss:** Mean profit of winners vs mean loss of losers
- **Largest Win/Loss:** Best and worst single trades

### Equity Curve
- **X-axis:** Time (dates)
- **Y-axis:** Account value
- **Features:**
  - Hover tooltip: Date, equity, % from start
  - Toggle linear/log scale
  - Zoom controls for date range
  - Highlight drawdown periods in red shading

### Drawdown Chart
- **Shows:** % decline from equity peak over time
- **Identifies:** Periods of sustained losses
- **Max Drawdown:** Annotated with arrow/label

### Monthly Returns Heatmap
- **Grid:** Rows = years, Columns = months
- **Color-coded:** Green (profit), Red (loss), Yellow (break-even)
- **Hover:** Shows exact % return for month

### Trade Distribution Histogram
- **X-axis:** Return buckets (-3% to +6%)
- **Y-axis:** Number of trades in each bucket
- **Insight:** Visualize if wins are larger/more frequent than losses

### Trade Log Table
- **Columns:**
  - Date (entry date)
  - Type (LONG/SHORT)
  - Entry Price
  - Exit Price
  - Shares
  - P&L ($)
  - Return (%)
  - Win/Loss indicator (ðŸŸ¢ðŸ”´)
- **Features:**
  - Sortable columns
  - Filter: Winners only, Losers only, Date range
  - Pagination (show 10/25/50 per page)
  - CSV export button

### Additional Statistics
- **Avg Trade Duration:** Hours held per trade
- **Win/Loss Streaks:** Longest consecutive wins/losses
- **Best/Worst Hours:** Performance by time of day
- **Costs:** Commissions and slippage impact
